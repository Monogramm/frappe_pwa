How to install

Learn how to install this application on your device

This application is actually a Progressive Web App (PWA) that can be directly installed to your device. Learn more about how to install this application on your device:

Install on iOS

To install a PWA on your Home Screen, follow these steps:
  • Tap the 'Share' button of 'Safari'
  • Tap the icon labeled 'Add to Home Screen'
  • Tap 'Add' in the upper-right corner
  • Confirm the name of the PWA then tap 'Add' in the upper-right corner
  • You can now launch the application from your Home Screen

Install on Android

Using Chrome for Android, you will automatically be prompted to install the app on your system. Click 'Install' to install the application on your device.

Install on Chrome (desktop)

When using Chrome for Desktop, you can click on the menu (the 3 dots in the upper-right corner), then 'Install ...' to add the application to Chrome Applications.
